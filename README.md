# 🔌 C Embedded Systems Playground

A structured C project focused on **embedded systems concepts**, **electronics-friendly programming**, and **low-level logic simulation**.  
This repository is designed to strengthen understanding of **bare-metal C**, **ADC logic**, and **microcontroller-style thinking** — without relying on heavy frameworks.

---

## 📌 Features

- Pure **C language** (no external libraries)
- Embedded-style coding patterns
- ADC & sensor logic simulation
- Hardware-agnostic design
- Clean, readable, and well-commented code
- Suitable for **ESP32 / STM32 / AVR mindset**
